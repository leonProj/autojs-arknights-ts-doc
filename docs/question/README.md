# 疑问解答
## 能否在手机上使用？
此项目目的是为了在pc模拟器上使用。尚未做手机端的适配，模拟器开发也方便。等项目稳定后，会考虑做手机端的适配。或者可以自己二开。
## 一定要安卓7.0及其以上的模拟器？
无障碍服务的点击，滑动等需要安卓7.0版本以上。否则需要开启root权限或adb权限，太麻烦了。
[详情见这](https://pro.autojs.org/docs/zh/v8/coordinatesBasedAutomation.html)
